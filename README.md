# helloSpringMVC
Spring MVC를 활용하여 만든 웹 응용 프로그램 과제</br>
MySQL DB에 현재까지 수강한 연도, 과목이름, 교수님 성함, 학점 정보를 저장
사용자 계정 DB도 만들어 인증 후 접속하도록 함.

## 기능
1.학기 별 이수 학점 조회</br>
2.수강 신청하기</br>
3.수강 신청 조회 메뉴</br>
